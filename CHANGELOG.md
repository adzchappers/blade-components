## 1.0.0 - 2026-05-28

### Added

Components:

- x-form: CSRF output, method spoofing (PUT/PATCH/DELETE), file upload
- x-form-fieldset: grouped inputs with optional legend and disabled state
- x-form-input: text/email/password/file/hidden with label and error
- x-form-textarea: multiline input with label and error
- x-form-select: single and multi-select with label and error
- x-form-checkbox: single checkbox with correct unchecked old-input handling
- x-form-radio: single radio button
- x-form-label: standalone label with optional required indicator
- x-form-button: submit/button/reset with translatable default label
- x-form-error: inline field error message
- x-form-error-list: full error summary list

**Full Changelog**: https://github.com/adzchappers/blade-components/commits/1.0.0
